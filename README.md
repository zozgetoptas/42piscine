## üéØ Module Details

### **c00: Basic C Fundamentals**
Introduction to the C language with focus on character output and basic control flow.

**Key Functions:**
- `ft_putchar()` - Output single characters
- `ft_print_alphabet()` - String iteration
- `ft_print_numbers()` - Loop control
- `ft_is_negative()` - Conditional logic
- `ft_print_comb()` - Nested loops & number combinations

### **c01: Pointers & Memory**
Deep dive into pointer concepts and array manipulation.

**Key Concepts:**
- Variable addresses and pointer dereference
- Function parameters and pass-by-reference
- Array manipulation with pointers
- Integer division and modulo operations

**Key Functions:**
- `ft_ft()` - Pointer dereferencing
- `ft_swap()` - Variable swapping
- `ft_div_mod()` - Division and modulo
- `ft_putstr()` - String output using pointers
- `ft_strlen()` - String length calculation
- `ft_sort_int_tab()` - Bubble sort implementation

### **c02: String Processing**
String manipulation and character classification functions.

**Key Functions:**
- `ft_strcpy()` - String copying
- `ft_strncpy()` - Safe string copying with limit
- `ft_str_is_alpha()` - Alphabetic character validation
- `ft_str_is_numeric()` - Numeric validation
- `ft_str_is_lowercase()` / `ft_str_is_uppercase()` - Case checking
- `ft_strupcase()` / `ft_strlowcase()` - Case conversion
- `ft_strcapitalize()` - Capitalization logic
- `ft_strlcpy()` - Safe string copying with return value

### **c03: Advanced String Operations**
Complex string manipulation and pattern matching.

**Key Functions:**
- `ft_strcmp()` - String comparison
- `ft_strncmp()` - Limited string comparison
- `ft_strcat()` - String concatenation
- `ft_strncat()` - Safe concatenation
- `ft_strstr()` - Substring search (pattern matching)
- `ft_strlcat()` - Safe concatenation with size limit

### **c04: Input & Parsing**
String conversion and number parsing algorithms.

**Key Functions:**
- `ft_strlen()` - String length
- `ft_putstr()` - String output
- `ft_putnbr()` - Integer output with sign handling
- `ft_atoi()` - ASCII to Integer conversion (handles signs and spaces)

### **c05: Recursion & Algorithms**
Mathematical algorithms implemented both iteratively and recursively.

**Key Functions:**
- `ft_iterative_factorial()` / `ft_recursive_factorial()` - Factorial calculation
- `ft_iterative_power()` / `ft_recursive_power()` - Power operation
- `ft_fibonacci()` - Fibonacci sequence generation
- `ft_sqrt()` - Integer square root (O(‚àön) algorithm)
- `ft_is_prime()` - Prime number validation
- `ft_find_next_prime()` - Next prime number finder

### **c06: Command-line Arguments**
Processing and manipulating program arguments.

**Key Functions:**
- `ft_print_program_name()` - Accessing program name
- `ft_print_params()` - Argument enumeration
- `ft_rev_params()` - Reverse argument order
- `ft_sort_params()` - Argument sorting

### **c07: Dynamic Memory & Advanced Techniques**
Memory allocation and complex string operations.

**Key Functions:**
- `ft_strdup()` - Dynamic string duplication (malloc)
- `ft_range()` - Dynamic array creation
- `ft_ultimate_range()` - Enhanced range with error handling
- `ft_strjoin()` - Multi-string concatenation with memory management
